# Template Files

This directory contains standardized templates for product documentation.

## Available Templates

### Business Requirements
- [`brd-template.md`](../../templates/brd_template.md) - Business Requirements Document
  - Executive summary and ROI
  - Strategic alignment and success metrics
  - High-level requirements and go-to-market strategy

### Product Requirements
- [`prd-template.md`](../../templates/prd_template.md) - Product Requirements Document
  - Detailed user stories and acceptance criteria
  - Functional and non-functional requirements
  - Design specs and technical implementation details

### Initiative Planning
- [`epic-template.md`](../../templates/epic_template.md) - Epic Template
  - High-level objectives and hypothesis
  - Phasing and user stories
  - Success metrics and dependencies

## Template Structure

All templates follow the placeholder format:
- `[PLACEHOLDER_NAME]` - Values to be filled in
- `<!-- Example: ... -->` - Examples showing how to complete each section

## Usage

1. Choose the appropriate template for your document type
2. Copy the template content
3. Replace all `[PLACEHOLDERS]` with actual values
4. Reference examples in comments for guidance
5. Remove placeholder comments before finalizing
